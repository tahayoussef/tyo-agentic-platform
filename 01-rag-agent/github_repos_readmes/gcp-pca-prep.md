# gcp-pca-prep

> Study notes and documentation for the **Google Cloud Professional Cloud Architect (PCA)**
> certification.

## Overview

`gcp-pca-prep` is a **documentation-only** repository collecting structured study material
for the **Google Cloud Professional Cloud Architect** exam. It is not application code — it
is a personal knowledge base of exam-domain notes, case-study breakdowns, decision
frameworks, and quick-reference cheat sheets used to prepare for and pass the certification.

## Contents

- **Exam domain notes** – compute (GCE, GKE, Cloud Run, App Engine), storage & databases
  (GCS, Cloud SQL, Spanner, Bigtable, Firestore, BigQuery), networking (VPC, load balancing,
  hybrid connectivity), security & IAM, operations (monitoring, logging), and cost
  optimization.
- **Case study analyses** – worked breakdowns of the official PCA case studies (e.g. EHR
  Healthcare, Helicopter Racing League, Mountkirk Games, TerramEarth), mapping business
  requirements to GCP solutions.
- **Decision frameworks** – "which compute / which database / which storage class" selection
  guides and trade-off tables.
- **Cheat sheets** – service comparisons, quotas and limits, and exam-day reminders.

## Format

Pure Markdown documentation organized by exam domain and case study, intended for
spaced-repetition review rather than execution.
